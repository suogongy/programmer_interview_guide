# programmer_interview_guide
程序员面试指南

## 📖 关于本指南

本指南专为2025年程序员求职面试而设计，涵盖了当前技术栈、新兴技术趋势以及完整的面试准备流程。无论您是应届毕业生还是有经验的开发者，都能在这里找到有用的指导和资源。

## 📅 学习计划建议

### 3个月冲刺计划
- **第1个月**: 基础算法 + 核心技术栈复习
- **第2个月**: 系统设计 + 项目实战
- **第3个月**: 模拟面试 + 查漏补缺

### 6个月全面准备
- **前2个月**: 技术基础强化
- **第3-4个月**: 高级主题学习
- **第5-6个月**: 实战项目 + 面试练习
